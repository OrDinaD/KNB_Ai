# Инструкция по развертыванию для демонстрации на конференции (2026)

Для успешного выступления и защиты проекта СНТК по философии, необходимо обеспечить доступ к сайту для всех участников в зале (ориентировочно 30 человек).

## 1. Локальный запуск (Production Mode)

Для максимальной производительности и отсутствия задержек при демонстрации запустите проект в режиме сборки:

```bash
# 1. Сборка проекта
npm run build

# 2. Запуск сервера (на порту 3000 по умолчанию)
npm run start
```

## 2. Настройка Cloudflare Tunnel

Согласно технологическому стандарту 2025-2026 (PDF стр. 13), Cloudflare Tunnel является наиболее стабильным и бесплатным способом проброса локального порта в интернет.

### Шаг 2.1: Установка cloudflared
Если у вас macOS и установлен Homebrew:
```bash
brew install cloudflared
```

### Шаг 2.2: Создание туннеля
Авторизуйтесь в Cloudflare и создайте туннель (рекомендуется через консоль или панель Zero Trust):
```bash
cloudflared tunnel login
cloudflared tunnel create philosophy-demo
```

### Шаг 2.3: Проброс порта (Quick Demo Mode)
Если вам нужна мгновенная ссылка без настройки домена:
```bash
cloudflared tunnel --url http://localhost:3000
```
**Cloudflare выдаст временную ссылку вида `https://random-words.trycloudflare.com`.**

## 3. Подготовка QR-кода
1. Скопируйте полученную ссылку.
2. Сгенерируйте QR-код и выведите его на последний слайд презентации.
3. Участники сканируют код и попадают на ваш локальный ноутбук.

## 4. Сбор и анализ данных
После 10-15 минут игры вы можете остановить сервер и проанализировать файл `telemetry.log` в корне проекта. 
Этот файл содержит неопровержимые доказательства детерминизма:
- По мере игры точность предсказаний ИИ (AI WON) должна расти.
- Процент ничьих (DRAW) должен снижаться по мере обучения матрицы сессии.
